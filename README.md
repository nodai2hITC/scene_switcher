# SceneSwitcher

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'scene_switcher'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install scene_switcher

## Usage

```ruby
switch_to <filename> <priv=false>
```
